## Project description

The project is a BSc thesis conducted under the supervision of Prof. Wiesław Babik at Jagiellonian University, Kraków, Poland. The main goals of the performed analysis are:

1. **Identification and annotation** of the all Toll-like receptors (TLR) sequences in *Pleurodeles waltl* genome (BLAST-p)  
2. **Expression analysis (RNA-seq)** of identified TLR genes in different ontogenetic stages and tissues
3. **Standardization of expression** from the raw data to FPKM and TPM units 
4. **Visualisation** of the FPKM and TPM data into relevant plots by ggplot2.

The outcome of the analysis is going to be discussed in the final form of the thesis' chapters, which are included in the **documents** section. Visualised data is going to be comparised with the evo-devo hypothesis, as it states "In the amphibian’s freshwater ontogenetic stages, expression of the “fish” TLR is higher than in the terrestrial adult".
